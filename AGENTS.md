# AGENTS.md

## このリポジトリの目的

このリポジトリは、GitHub Issue駆動のAI実装教育用sandboxです。

受講者が、AIを使ってIssueを実装し、PR、レビュー、CI、修正、マージ判断まで学ぶための練習環境です。

## 基本ルール

- 日本語で説明する。
- 1 Issue = 1 Branch = 1 PR を基本にする。
- Issue外の変更をしない。
- 実装前にDoDを確認する。
- 変更後は `npm test` を実行する。
- PR本文には、変更内容、検証結果、リスク、DoD充足状況を書く。

## 禁止事項

- `.env*` を作成、閲覧、編集しない。
- APIキー、認証情報、個人情報、実顧客情報を扱わない。
- 本番API、本番DB、外部送信を扱わない。
- 金銭判断、契約変更、カード操作、振込操作を扱わない。
- mainへ直接pushしない。
- Issue外の親切修正を追加しない。

## AIツールの使い分け

- Codex: 基本の実装、テスト追加、PR修正、レビュー対応。
- Claude Code: 複雑な計画、広範囲調査、サブエージェント、スキル活用。
- Antigravity: 補助教材。ログイン情報はDiscordで千葉に確認する。

## 完了条件

作業完了時は以下を確認する。

- IssueのDoDを満たした。
- テストが通った。
- 差分を説明できる。
- 不要な変更がない。
- 禁止事項に触れていない。
