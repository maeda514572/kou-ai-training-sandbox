# Antigravityの教科書

Antigravityの教科書は、このロードマップでは補助教材として扱います。

## 位置づけ

- 初期必修ではありません。
- Lv3以降で、Plan、DoD、artifact-first、検証証跡を理解する補助教材として使います。
- Lv5以降で、複数エージェントや大きめ設計の理解に使います。

## ログイン情報

ログイン情報は、Discordで千葉に聞いてください。

## 読むときの観点

- IssueからPRまでの流れと接続できるか
- Planや検証証跡の残し方に使えるか
- Codex / Claude Code と役割が重複していないか
- 初学者にとってツールが増えすぎないか
