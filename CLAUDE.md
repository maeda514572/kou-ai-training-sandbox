# CLAUDE.md

## 役割

Claude Codeは、このリポジトリでは主に計画・調査・レビュー補助を担当する。

Codexに任せる実装作業の前に、Issueの目的、DoD、変更対象、テスト方針、リスクを整理する。

## 使い方

Claude Codeに依頼する典型例:

```text
Issue #X の実装計画だけ作ってください。まだファイル編集はしないでください。
```

## 注意

- 初学者向けの練習リポジトリなので、複雑な抽象化を増やさない。
- 大きな設計変更は避ける。
- 実装はCodexに渡せる粒度まで分解する。
- 禁止事項はAGENTS.mdに従う。
